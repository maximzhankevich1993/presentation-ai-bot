# Telegram бот Презентатор ИИ

Бот для генерации презентаций с автоопределением страны и конвертацией цен в локальную валюту.

## Команды

| Команда | Описание |
|:---|:---|
| `/start` | Приветствие с ценами в локальной валюте |
| `/new` | Создать презентацию |
| `/plans` | Тарифы (конвертируются по стране) |
| `/vip` | Активировать VIP-доступ |
| `/referral` | Реферальная программа |
| `/help` | Помощь |

## Запуск

```bash
npm install
BOT_TOKEN=твой_токен node server.js
